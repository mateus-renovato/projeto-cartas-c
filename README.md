# Projeto De Cartas C
Projeto desenvolvido em linguagem C para a faculdade, com foco na criação de cartas e comparação de atributos entre elas. O objetivo é praticar lógica de programação, estruturas e tomada de decisão.

# Comparação de Cartas em C

Projeto desenvolvido como atividade da faculdade.

## 📌 Objetivo
Simular cartas com atributos e realizar a comparação entre elas, utilizando a linguagem C.

## 🛠 Tecnologias
- Linguagem C

## ▶️ Como executar
1. Compile o arquivo:
   gcc Cartas.c -o cartas
2. Execute:
   ./cartas

## 📚 Aprendizados
- Estruturas de dados
- Comparações condicionais
- Lógica de programação em C
